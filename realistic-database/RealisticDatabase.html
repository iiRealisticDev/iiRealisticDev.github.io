# new RealisticDatabase(dbDir, dbFile, options)

Create a database.

Creates a database at the given path, and presents the necessary methods to interact with it.

Parameters:
Name Type Description
dbDir string

A path to the directory the DB should be in.

A file where the DB should be stored. Must be JSON.

A list of DB options.
